## Fabrizio Guccione

**Embedded Intelligence Designer** · NYU ITP MPS '26

I build AI-driven interactive systems that live in physical space — not just on screens. My work bridges machine learning, physical computing, and fabrication to create things that probably shouldn't exist yet.

Currently running ITP's Fabrication Shop. Previously McCann NY, ADEAM, NYU CREATE Lab.
